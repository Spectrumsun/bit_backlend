# bit-backend

A simple App in Javascript, Node, and express

# View App
  * Host App Backend  https://bit-60by.onrender.com/api/v1/

# Technologies Used
  * MYSQL
  * Express
  * Javascript
  * NodeJS

  #Endpoints
  * GET: All state  https://bit-60by.onrender.com/api/v1/states
  * GET: Local govt result  https://bit-60by.onrender.com/api/v1/lga-results
  * GET: Wards  https://bit-60by.onrender.com/api/v1/wards
  * GGT: All Lga  https://bit-60by.onrender.com/api/v1/lga
  * POST: Create new polling unit  https://bit-60by.onrender.com/api/v1/new-unit


# Run Locally
  * Download or clone the repo
  * Make sure you have the latest `node` and `npm` installed on your system
  * Open the terminal inside the root directory of the repo
  * Run `npm install` or `npm i` to install all dependencies
  * Make sure your MYSQL data is running 
  * Run the database migration
  * Add the datebase information to the .env file
  * Run `npm start` to start the in development
  * Go to `127.0.0.1:6000/api/v1`
